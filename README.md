# Nutri-React

Projeto desenvolvido para estudos utilizando **React**.

## 🚀 Tecnologias

- JavaScript
- React
- Node
- HTML
- CSS
 

## ⚙️ Como executar o projeto

Clone o repositório

- git clone https://github.com/seu-usuario/seu-repositorio.git

Acesse a pasta do projeto

- cd nutri-react

Instale as dependências

- npm install

Execute o projeto

- npm start

O projeto será iniciado em:

http://localhost:3000
